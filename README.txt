Steps to predict the flower type:
1. Type or browse the flowers directory path. The given path should include inside a directory with the flowers in it( given_path\directory\flowers_images.jpg ).
2. Type or browse the model path(flowers_model.h5).
3. Load the model by pressing the "Load Model" button.
4. Activate the model by pressing the "Predict" button.
5. Choose the wanted directory to save the results(csv type) file.
A window with the results will now open.
